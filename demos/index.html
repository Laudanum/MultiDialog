MultiDialog can handle different content types.

A group of mixed content types with loop functionality enabled.


  • Images (jpg, png, gif by file extension)
  • AJAX content (by adding ?ajax=true to the link)
  • iframes (by adding ?iframe=true to the link)
  • You Tube video (by domain)
  • Vimeo video (by domain)
  • Inline content (by anchor link)
			$( "a.group" ).MultiDialog({
				gallery: {
					enabled: true,
					loop: true
				}
			});

This example explains how to use the jQuery UI Dialog options.

Open a bigger, resizable, draggable, top positioned Dialog without an overlay.


  • You Tube video
  • HTML
  • Image
			
			$( "a.dialog-options" ).MultiDialog({
				dialog: { 
					width: 900,
					height: 600,
					modal: false,
					position: {
						my: 'top',
						at: 'top',
						offset: '0 10'
					},
					draggable: true,
					resizable: true
				}
			});
Please note you need to add jquery.ui.draggable.js and jquery.ui.resizable.js.

How to add a path related custom content type.

Demo link 
	
			$( "a.add-type" ).MultiDialog({
				types: {
					config: {
						customPath: {
							test: function( href ) {
								// search for syntaxhighlighter in URL
								return href.match( /syntaxhighlighter/i );
							},					
							template: ''
						}
					}
				},
				openCustom: function( data ){
					if ( data.type = "customPath" ) {
						this._parseHtml( data, "iframe", "url" );
						data.width = 1280;
						data.height = 800;
						data.title = "SyntaxHighlighter";
						this._open( data );
					} else {
						// backfall
						window.open( data.href, '_newtab' );
					}
				}
			});
